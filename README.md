# Project 1: Model Evaluation

This project was completed during my internship at Hex Softwares.

## Objective
To evaluate the performance of a classification model using the Iris dataset.

## Tasks Performed
- Split the dataset into training and testing sets
- Trained a classification model
- Evaluated the model using accuracy, precision, and recall
- Visualized results using a confusion matrix
- Generated a classification report

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Google Colab
